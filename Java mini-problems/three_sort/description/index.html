Ταξινόμηση πίνακα ακεραίων από 0, 1, 2.

Σύνοψη

Σε αυτή την άσκηση πρέπει να γράψετε κώδικα που θα ταξινομεί έναν πίνακα ακεραίων με τις τιμές 0, 1, 2 σε γραμμικό χρόνο.

Περιγραφή

Στο πρόγραμμα θα συμπληρώσετε την μέθοδο threeSort.

Η μέθοδος πρέπει να ταξινομεί έναν πίνακα ακεραίων (IntegerArray) σε αύξουσα σειρά.

Ο πίνακας περιέχει μόνο τους αριθμούς 0, 1 και 2.

Για την ταξινόμηση χρησιμοποιείστε μόνο τις μεθόδους length, compare και swap της κλάσης IntegerArray.

  • length(): επιστρέφει το μέγεθος του πίνακα
  • compare(i, x): συγκρίνει το στοιχείο i του πίνακα με τον αριθμό x και επιστρέφει θετικό αριθμό αν ο x είναι μεγαλύτερος, αρνητικό αν ο x είναι μικρότερος και μηδέν αν είναι ίσοι
  • swap(i, j): αλλάζει τη θέση των στοιχείων i και j του πίνακα

Ο αλγόριθμος που θα αναπτύξετε θα πρέπει να κάνει την ταξινόμηση σε γραμμικό χρόνο.
